#  House Price Prediction using Machine Learning

##  Project Overview

This project predicts house prices based on features like area, number of bedrooms, bathrooms, and location using machine learning techniques. It helps in estimating property prices accurately.

---

##  Objective

* Analyze housing data
* Identify important features affecting price
* Build a predictive model using regression

---

##  Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## Dataset

The dataset includes:

* Area (sq ft)
* Bedrooms
* Bathrooms
* Location
* Price

⚠️ Dataset is not included due to size.

## ▶️ How to Run

1. Download the dataset from the link above
2. Place `house_data.csv` inside the project folder
3. Run the program:

```bash
python app.py
```
