About This Project

📌 Project Description

The House Price Predictor is an AI-powered web application that helps users estimate property values based on key features such as area, number of bedrooms and bathrooms, location, and age of the house.

🎯 Purpose

This application aims to provide quick and reliable house price estimates to help:

  • Home buyers make informed purchasing decisions
  • Property sellers set competitive prices
  • Real estate agents provide instant valuations
  • Investors evaluate potential property investments

✨ Key Features

🤖 Machine Learning

Random Forest Regressor algorithm for accurate predictions

💰 Cost Breakdown

Detailed calculation including GST, stamp duty, and registration

📱 Responsive Design

Works seamlessly on desktop, tablet, and mobile devices

📧 Contact Form

Easy enquiry submission and CSV storage
